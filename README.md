## 🚀 セットアップ方法

### 必要な環境
- モダンなウェブブラウザ（Chrome, Firefox, Safari, Edge等）
- ローカルサーバー（推奨）

### 起動方法

#### 方法1: ブラウザで直接開く
1. `index.html` をブラウザにドラッグ&ドロップ

#### 方法2: ローカルサーバーを使用（推奨）

**Python を使用する場合:**
```bash
# Python 3.x
python -m http.server 8000

# Python 2.x
python -m SimpleHTTPServer 8000
```

**Node.js (http-server) を使用する場合:**
```bash
# インストール
npm install -g http-server

# サーバー起動
http-server -p 8000
```

**VS Code の Live Server 拡張機能を使用する場合:**
1. Live Server 拡張機能をインストール
2. `index.html` を右クリック
3. "Open with Live Server" を選択

ブラウザで `http://localhost:8000` にアクセスしてください。

## 🎯 カスタマイズ方法

### カラーの変更
[css/common.css](css/common.css) の `:root` セクションでカラー変数を変更してください。

```css
:root {
  --primary-color: #00BCD4;      /* プライマリカラー */
  --secondary-color: #E91E63;    /* セカンダリカラー */
  --accent-color: #9C27B0;       /* アクセントカラー */
  /* ... */
}
```
